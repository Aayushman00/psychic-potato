# Project Documentation

This directory contains all research and design documentation for the Piano‑to‑Keyboard Mapping Optimizer.

## Structure

- `research/` – Narrative of the research process, literature review, design evolution, mathematical formulation, and open questions.
- `architecture/` – High‑level system architecture, component diagrams, data flow, and key design decisions (ADRs).

Start by reading `research/00-project-overview.md` for a high‑level summary, then proceed through the numbered research files in order.
