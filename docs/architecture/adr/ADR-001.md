# ADR-001: Replace penalty‑based objective with graph distortion (QAP)

**Status**: Accepted

**Context**
The initial prototype used a penalty‑based objective function that summed terms such as placement penalty, adjacency penalty, octave spread, finger strain, hand balance, modifier usage, ghosting, stretch, and topology, each with manually tuned weights. Empirical evaluation showed that the optimizer would often collapse notes onto a few easy‑to‑reach keys to minimise the sum of penalties, destroying musical relationships. Moreover, the weights lacked a principled foundation and required tedious trial‑and‑error tuning.

**Decision**
Adopt a Quadratic Assignment Problem (QAP) formulation:
- Define a weighted undirected graph **Gₚ = (Vₚ, Eₚ, W)** for the piano, where vertices are notes and edge weight W_ij reflects the importance of preserving the relationship between notes i and j.
- Define a weighted directed graph **Gₖ = (Vₖ, Eₖ, C)** for the keyboard, where vertices are keyboard states (key + hand + finger + modifier) and edge weight C_ab is the predicted movement time/effort to go from state a to b.
- Seek an injective (or bijective) mapping f: Vₚ → Vₖ that minimises  
  \[
  \sum_{i,j \in V_p} W_{ij} \cdot C_{f(i),f(j)} .
  \]
This directly couples musical importance with biomechanical cost, eliminating the need for ad‑hoc penalty weights.

**Alternatives Considered**
1. **Retain penalty terms with data‑driven weights** – still relies on hand‑crafted basis functions and requires regression to learn weights; less transparent.
2. **Spectral relaxation / eigen‑embedding** – drops integrality, making post‑processing difficult to guarantee a valid assignment.
3. **Linear sum assignment (LSA)** – only captures unary costs, cannot model pairwise relationships essential for musical interval preservation.
4. **Keep original penalty set but derive weights from first principles** – would still accumulate many terms; combining them into a single quadratic form is more elegant.

**Consequences**
- **Positive**: The objective now has a clear theoretical grounding (graph distortion). All constants in W and C arise from first‑principles models of auditory perception and biomechanics. The formulation encourages preservation of musically important intervals while discouraging inefficient movements.
- **Negative**: The problem is NP‑hard; exact solution is infeasible for 88 notes, necessitating a heuristic (chosen to be Simulated Annealing – ADR‑004). Implementation must ensure the cost matrices are correctly built and the optimisation respects the injectivity constraint.
- **Implementation Impact**: Updating the codebase to replace the penalty module with graph construction and QAP objective; introducing matrix multiplication–based evaluation; adding ADR documentation.
