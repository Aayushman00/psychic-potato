# Keyboard Layout Optimizer

A pure Python optimization engine for mapping piano notes to QWERTY keyboard keys with ergonomic scoring and experiment reporting.

## Structure

- `keyboard_layout_optimizer/` - core package
- `configs/` - YAML configuration files
- `exports/` - generated experiment outputs
- `experiments/` - experiment definitions
- `tests/` - unit tests

## Running

Install dependencies and run the main experiment:

```powershell
pip install pyyaml matplotlib pytest
python main.py
```
